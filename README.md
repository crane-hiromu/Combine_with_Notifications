# Combine_with_Notifications

## Description

Handle rotating from NotificationCenter with Combine.

You can choose 3 pattern.
- UIViewController + MVVM
- SwiftUI + MVVM (output from AnyPublisher)
- SwiftUI + MVVM (output from @Published)
